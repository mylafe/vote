#### vote

基于yii2-advanced在线投票

#### 配置

    前台：vote/frontend/web
    
    后台：vote/backend/web

#### 功能

后台：

- 数据统计
- 用户管理
- 投票管理
- 权限控制

前台：

- 投票浏览
- 投票防刷
- 搜索
- 个人信息
